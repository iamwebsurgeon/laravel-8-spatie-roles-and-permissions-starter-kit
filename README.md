# laravel-8-spatie-roles-and-permissions-starter-kit
laravel-8-spatie-roles-and-permissions-starter-kit If you are looking for ready to use multiple roles and permissions system, then you are at the right place. Follow the simple steps.
1. Simple download the repository
2. Make a database with the name of your choice
3. Change database name in the .env file according to your database name
4. run php artisan migrate command
5. run php artisan serve

You can skip step 4 and import the provided database. Hope you like it and best of luck for your new project.
